## Installation & Set Up

1. Clone this repository
```bash
git clone https://github.com/yuvikaKathaith/PortfolioWebsite_v1.git
```

2. Navigate to the project directory
```bash
cd PortfolioWebsite_v1
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```
